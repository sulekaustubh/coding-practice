âœ… What is CORS ?
---------------------------------------------------------------------------
   â— CORS (Cross-Origin Resource Sharing)
   â— It is a security feature implemented by web browsers
   â— It helps to control cross-origin requests to
     prevent unauthorized access to resources
   â— Without CORS, web pages could make requests to any domain,
     thus exposing sensitive data and creating security vulnerabilities
   â— CORS ensures security of a web app by 
     enforcing a same-origin policy by default
   â— When a web page in one origin tries to make an HTTP request to a different origin, 
     the browser blocks the request unless the target server explicitly allows it

âœ… What is Virtual DOM in React ?
---------------------------------------------------------------------------
   â— It is a virtual representation of the actual DOM
   â— It is a lightweight in-memory data structure that React 
     uses to optimize the process of updating the actual DOM
   â— It improves performance by minimizing the number of 
     direct manipulations to the real DOM

âœ… How does Virtual DOM actually work ?
---------------------------------------------------------------------------
   â— React maintains 2 states of the VDOM in its memory
     1. Previous state
     2. Updated state
   â— The Diffing algorithm calculates the difference between these 2 states
   â— The resultant difference is applied as a change 
     in the Real DOM in the most efficient manner
   â— This process is also known as Reconciliation
